9) На HQ-SRV настройте программное обеспечение fail2ban для защиты ssh

apt-get install fail2ban python3-module-systemd -y

sed -i 's/before = paths-altlinux.conf/before = paths-altlinux-systemd.conf/' /etc/fail2ban/jail.conf

===
cat << EOF > /etc/fail2ban/jail.d/sshd.conf
[sshd]
enabled = true
port = 2026
filter = sshd
backend = systemd
maxretry = 3
bantime = 1m
EOF
===

systemctl enable --now fail2ban

systemctl status fail2ban.service

fail2ban-client status sshd

Проверку можно выполнить с HQ-CLI (CLI)

ssh -p 2026 sshuser@hq-srv

3 раза вводим не верный пароль

смотрим логи на hq-srv командой

fail2ban-client status sshd
